Example Content Ideation Prompts

Blog Series Planning Prompt Example

Develop a comprehensive 12-part blog series plan on "sustainable home renovation" targeting homeowners interested in eco-friendly remodeling. For this series:

  • Create an overarching theme that ties all 12 posts together
  • Provide specific titles for each post that incorporate relevant keywords
  • Include a brief outline of key points to cover in each post (5-7 bullet points)
  • Suggest internal linking opportunities between posts
  • Recommend a logical publishing order that builds reader knowledge progressively
  • Include ideas for multimedia elements that would enhance each post
  • Suggest call-to-action ideas appropriate for each post topic

The series should cover both beginner concepts and more advanced techniques, with a focus on practical, actionable advice. Each post should be optimized for both search engines and reader engagement.

Pillar-Cluster Structure Prompt Example

Create a comprehensive pillar-cluster content strategy for a website focused on "digital marketing for small businesses." Develop:

  • 5 main pillar page topics with compelling title suggestions and primary keywords
  • For each pillar, suggest 8-10 cluster content topics with their own keywords
  • Provide a brief content outline for each pillar page (1500-2000 words)
  • Suggest internal linking strategies between pillars and clusters
  • Recommend content formats for different cluster topics (guides, case studies, templates, etc.)
  • Include suggestions for featured snippet optimization
  • Propose a content development sequence based on search volume and competition

The pillar-cluster model should create a comprehensive resource hub that establishes topical authority while addressing the full spectrum of small business digital marketing needs, from beginner concepts to advanced strategies.

FAQ Generation Prompt Example

Generate a comprehensive FAQ section for a website about "personal finance and investing for young professionals." Create:

  • 40-50 highly relevant questions organized into logical categories
  • For each question, provide a concise yet thorough answer (100-150 words)
  • Ensure questions reflect actual search queries people use (question-based keywords)
  • Include a mix of basic and advanced questions to serve different knowledge levels
  • Suggest 5-7 main FAQ categories for organizational structure
  • Identify questions with featured snippet potential
  • Include proper FAQ schema markup recommendations

The FAQs should address common concerns, misconceptions, and information needs of young professionals looking to improve their financial literacy and begin investing. Focus on questions that demonstrate genuine information needs and search intent.

Content Calendar Prompt Example

Create a 6-month strategic content calendar for a "health and wellness coaching" business targeting busy professionals. Include:

  • 4-5 content pieces per month with specific topic suggestions and titles
  • Primary and secondary keywords for each content piece
  • Content types/formats for each piece (blog post, video script, infographic, etc.)
  • Word count recommendations based on topic and competition
  • Publication timing considerations (seasonal relevance, trending topics)
  • Content clusters and internal linking opportunities
  • Repurposing suggestions for each major content piece

The calendar should follow a strategic progression that builds audience knowledge while addressing seasonal trends and timely topics. Include a mix of evergreen content and timely pieces, with recommendations for updating existing content when appropriate.

Semantic Topic Grouping Prompt Example

Develop a comprehensive semantic topic grouping strategy for a "pet care and nutrition" website. Create:

  • 10-12 main semantic topic groups based on related search concepts
  • For each semantic group, identify 15-20 related subtopics and keywords
  • Map the user journey through these semantic groups
  • Suggest content types that would best serve each semantic group
  • Provide internal linking recommendations between semantically related content
  • Identify entity relationships that should be emphasized for semantic SEO
  • Suggest taxonomy and URL structure that reinforces semantic relationships

The semantic grouping should create a comprehensive content architecture that helps search engines understand the relationships between topics while providing users with intuitive navigation between related concepts. Focus on creating true topical depth rather than just keyword variations.
