  1. Select the type of page you're optimizing from the dropdown menu

Example On-Page SEO Prompts

Meta Title Optimization Prompt Example

Create 10 SEO-optimized meta title options for a product page about "organic bamboo bed sheets" targeting eco-conscious consumers. For each title:

  • Include the primary keyword "organic bamboo bed sheets" naturally
  • Incorporate at least one of these secondary keywords: "cooling sheets," "sustainable bedding," "hypoallergenic"
  • Keep the length between 50-60 characters to avoid truncation in search results
  • Include a compelling benefit or unique selling point
  • Use power words that encourage clicks
  • Maintain brand voice that appeals to environmentally conscious consumers

For each title option, provide the exact character count and an explanation of why it would be effective for both SEO and click-through rate. Rank the titles in order of recommended effectiveness.

Meta Description Optimization Prompt Example

Generate 5 SEO-optimized meta descriptions for a blog post titled "10 Essential Digital Marketing Strategies for Small Businesses in 2025." Each description should:

  • Include the primary keyword "digital marketing strategies for small businesses" naturally
  • Incorporate at least one secondary keyword from: "affordable marketing," "local business marketing," "ROI-focused"
  • Be between 140-155 characters to avoid truncation in search results
  • Include a clear value proposition for the reader
  • Contain a compelling call-to-action
  • Be written in an authoritative yet approachable tone

For each description, provide the exact character count and highlight which aspects would appeal to small business owners looking to improve their marketing efforts with limited resources.

Heading Structure Optimization Prompt Example

Create a comprehensive heading structure (H1-H4) for a long-form guide about "home workout routines for beginners" targeting people new to fitness. The heading structure should:

  • Include an SEO-optimized H1 that incorporates the primary keyword
  • Provide 5-7 H2 headings that cover the main subtopics and incorporate secondary keywords
  • For each H2, suggest 2-4 H3 subheadings that break down the content further
  • Where appropriate, include H4 headings for detailed sections
  • Ensure a logical content flow that guides beginners from basic concepts to implementation
  • Incorporate question-based headings where appropriate for featured snippet opportunities

For each heading, briefly explain its purpose in the overall content structure and how it supports both user experience and SEO goals. Indicate which headings have the highest potential to rank for specific search queries.

Image Alt Text Optimization Prompt Example

Generate SEO-optimized alt text for 8 product images on an e-commerce page selling "handcrafted leather wallets." For each image, assume the following types:

  • Main product image showing the full wallet
  • Close-up of the leather texture and stitching
  • Image showing the wallet open with card slots visible
  • Image showing the wallet in a pocket (size reference)
  • Color variants of the wallet displayed together
  • Image showing the packaging/gift box
  • Lifestyle image of someone using the wallet
  • Infographic showing wallet dimensions and features

For each image, create alt text that naturally incorporates relevant keywords without keyword stuffing, accurately describes what's shown in the image, and follows accessibility best practices.

Internal Linking Strategy Prompt Example

Develop a comprehensive internal linking strategy for a blog post about "sustainable gardening practices" that will be published on a gardening website. The website already has the following content:

  • Beginner's Guide to Organic Gardening
  • 10 Best Composting Methods for Home Gardens
  • How to Choose Eco-Friendly Garden Tools
  • Water Conservation Techniques for Gardens
  • Natural Pest Control Without Chemicals
  • Seasonal Planting Guide for Vegetable Gardens
  • Soil Health: The Foundation of Sustainable Gardens

For the new "sustainable gardening practices" article, provide specific anchor text suggestions for internal links to existing content, explain the contextual relevance and SEO benefit of each link, and recommend natural link placement within the content flow.
