MCP stands for Model Context Protocol. In this finder, it refers to the underlying AI platforms, APIs, or model-serving infrastructure that provide access to various Artificial Intelligence models (like GPT-4, Claude 3, Gemini, etc.).

Think of MCPs as the "engine rooms" or "servers" that allow you to use specific AI capabilities for SEO tasks. Choosing an MCP involves considering factors like the available models, API costs, performance (speed/latency), context window size, ease of integration, and specific features they offer for managing interactions with the AI. Examples include the OpenAI API, Anthropic API, Google Vertex AI, and Hugging Face Inference Endpoints.
